Nubiot es una plataforma para el internet de las cosas, mediante nuestra API Restful puedes controlar tus dispositivos, también puedes monitorear datos enviados en tiempo real al dashboard.

Puedes controlar tu Arduino desde servicios como IFTTT ó Zapier para establecer eventos, por ejemplo: Encender un led rgb dependiendo el clima de tu ciudad, o notificarte cuando tengas un nuevo email, tuit, etc.

La API Restful de Nubiot te permite enviar eventos a tu Arduino o Esp8266 boards, con los cuales puedes controlar su comportamiento, puedes crear un sistema de notificaciones de una manera fácil y sencilla ó automatizar tu casa, Nubiot te brinda librerías para utilizar en plataformas como Arduino, Raspberry Pi y en Node.js.

Escoge tu tarjeta, tu app favorita y nubiot hará el resto!
